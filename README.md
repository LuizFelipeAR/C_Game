# 📖 Sobre o jogo

O projeto consiste em um sistema em **linguagem C** com três mini-games desenvolvidos em console:

1. **Perguntas e Respostas**
   Um jogo de quiz com 5 perguntas fixas, cada uma contendo 4 alternativas, sendo apenas uma correta.

2. **Cobra na Caixa 🐍**
   Cinco caixas são exibidas ao jogador.

   * Uma contém uma cobra ☠️
   * Outra contém um botão 🔘
   * As demais estão vazias

   Caso o jogador escolha:

   * a caixa com a cobra → **derrota**
   * a caixa com o botão → **vitória**
   * uma caixa vazia → a caixa permanece aberta, reduzindo as opções

3. **Gousmas War ⚔️**
   Mini-game de estratégia em turnos envolvendo criaturas chamadas Gousmas.

## ▶️ How to use ❓

1. Compile o arquivo principal em um compilador C
2. Execute o programa no terminal / console
3. Escolha um dos mini-games no menu principal
4. Siga as instruções exibidas na tela
5. Ao final, escolha entre jogar novamente ou retornar ao menu
